# World Happiness Report Analysis and Machine Learning Prediction

## Author
Junyuan Peng

## Project Description

This project analyzes the World Happiness Report dataset from 2015 to 2019 and explores the factors that influence happiness scores across different countries. Data visualization and machine learning techniques are used to identify important relationships and predict happiness scores.

## Dataset

The dataset contains information from the World Happiness Report (2015-2019), including:

- Happiness Score
- Economy (GDP per Capita)
- Family
- Health (Life Expectancy)
- Freedom
- Trust (Government Corruption)
- Generosity

## Project Objectives

- Analyze happiness scores across countries
- Identify factors related to happiness
- Visualize trends and correlations
- Build machine learning models to predict happiness scores

## Data Analysis

The following visualizations were created:

- Top 10 Happiest Countries
- GDP vs Happiness Score
- Family Support vs Happiness Score
- Correlation Heatmap
- Average Happiness Score by Year

## Machine Learning Models

### Linear Regression

R² Score: 0.7465

### Random Forest Regression

R² Score: 0.7934

The Random Forest model achieved better predictive performance than the Linear Regression model.

The analysis shows that GDP per capita, health, and family support are strongly associated with happiness scores. Machine learning models can effectively predict happiness levels using socioeconomic indicators.
